php 数组总结
2020-12-09 # php # 数组 # array

  • sort():按值升序排序 （健名重置）

  • rsort():按值降序排序 （健名重置）

  • ksort():按键名升序排序 （保留键名）

  • krsort():按健名降序排序 （保留键名）

  • asort():关联数组 键值 升序 （保留键名）

  • arsort():关联数组 键值 降序 （保留键名）

  • usort(): 自定义函数对数组进行排序 （键名重置）

  • uasort(): 自定义函数 键值排序 （保留键名）

  • uksort(): 自定义函数 键名排序

  • array_multisort(array1,sorting order,sorting type,array2,array3…): 对多个数组进行排序，可指定排序顺序和类型
    字符串键名将被保留，但是数字键名将被重新索引，数组类型必须相同

  • natsort():函数用”自然排序”算法对数组进行排序。键值保留它们原始的键名。(区分大小写)
    在自然排序算法中，数字 2 小于 数字 10。在计算机排序算法中，10 小于 2，因为 “10” 中的第一个数字小于 2

  • natcasesort():函数用”自然排序”算法对数组进行排序。键值保留它们原始的键名。(不区分大小写)
    在自然排序算法中，数字 2 小于 数字 10。在计算机排序算法中，10 小于 2，因为 “10” 中的第一个数字小于 2

  • shuffle(): 打乱数组 （键名重置）

  • ange(low,high,step): 创建指定返回的数组，可选 step 间隔

  • array_unique(): 返回 删除重复值的数组

  • array_values(): 返回 数组中的所有的值

  • array_keys()：返回 数组中的所有的键名

  • key(): 返回 数组当前指针指向的键名

  • current(): 返回 数组当前指向的值

  • next(): 指针下移，返回当前值

  • prev(): 指针上移，返回当前值

  • reset(): 指针移到首位，返回当前值

  • end(): 指针移到末尾，返回当前值
